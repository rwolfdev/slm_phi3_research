 ## Function Name: Addiere

### Description:
This function, `Addiere`, takes two integer parameters and returns their sum. It performs a simple addition operation on the two input integers and returns the result.

### Parameters:
1. `zahl1` (int): The first integer to be added.
2. `zahl2` (int): The second integer to be added.

### Return Type:
The function returns an integer, which is the sum of the two input integers.

### Function Syntax:
```csharp
public int Addiere(int zahl1, int zahl2)
{
    return zahl1 + zahl2;
}
```
