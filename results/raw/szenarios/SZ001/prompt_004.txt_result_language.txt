 # Function: Addiere

## Purpose
The `Addiere` function computes the sum of two integer values.

## Parameters
1. `zahl1` (int): The first integer value to be added.
2. `zahl2` (int): The second integer value to be added.

## Return Value
The function returns an `int` representing the sum of `zahl1` and `zahl2`.

## Syntax
```csharp
public int Addiere(int zahl1, int zahl2)
{
    return zahl1 + zahl2;
}
```

## Notes
- The function assumes that the inputs are valid integers.
