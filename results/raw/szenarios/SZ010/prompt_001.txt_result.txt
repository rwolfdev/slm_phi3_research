 # FullNameToColorConverter Class Documentation

## Overview

The `FullNameToColorConverter` class is a custom value converter that converts a full name string into a color based on the hash value of the name. This class is implemented in the `rwolfdev.gists` namespace and inherits from `IValueConverter`.

## Class Members

### Namespace

- `namespace rwolfdev.gists`

### Class

- `public class FullNameToColorConverter : IValueConverter`

### Methods

1. `Convert(object value, Type targetType, object parameter, CultureInfo culture)`

    - Converts a full name string to a color.
    - If the value is null, returns `Brushes.Black`.
    - Calculates the hash value of the name and converts it to a color using the `ColorScale.HslToRgb` method.

2. `ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)`

    - Returns the original value, as the conversion is not reversible.
