 # FullNameToColorConverter Class Documentation

## Purpose

The `FullNameToColorConverter` class is designed to convert a full name into a color based on the hash value of the name. It implements the `IValueConverter` interface to provide this conversion functionality.

## Class Members

### FullNameToColorConverter Class

- `Convert(object value, Type targetType, object parameter, CultureInfo culture)`: Converts the given value (full name) into a color.
  - `value`: The full name to be converted.
  - `targetType`: The target type of the conversion.
  - `parameter`: An optional parameter for the conversion.
  - `culture`: The culture information used for the conversion.

- `ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)`: Returns the original value (full name) as it is.
  - `value`: The value to be converted back.
  - `targetType`: The target type of the conversion.
  - `parameter`: An optional parameter for the conversion.
  - `culture`: The culture information used for the conversion.

### ColorScale Class

- `HslToRgb(double h_original, double s_original, double l_original)`: Converts an HSL color value to an RGB color value.
  - `h_original`: The original hue value.
  - `s_original`: The original saturation value.
  - `l_original`: The original lightness value.

## Public Methods

### Convert Method

The `Convert` method takes a full name as input and returns a `SolidColorBrush` object representing the color corresponding to the hash value of the name. If the input value is null, it returns a `Brushes.Black`.

### ConvertBack Method

The `ConvertBack` method returns the original full name value as it is, without any conversion.

## Public Properties

- `TargetType`: The target type of the conversion, which is `object`.
- `Parameter`: An optional parameter for the conversion, which is `null` by default.
- `Culture`: The culture information used for the conversion, which is `null` by default.

## Public Methods

### Convert Method

```csharp
public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
{
    if (value != null)
    {
        string name = value as string;

        int hash = 0;

        for (int i = 0; i < name.Length; i++)
        {
            int ascii = System.Convert.ToInt16(name[i]);
            hash = ascii + ((hash << 5) - hash);
        }

        hash = hash % 360;
        int h = hash % 360;
        int s = 45;
        int l = 60;

        return new SolidColorBrush(ColorScale.HslToRgb(h, s, l));
    }
    else
    {
        return Brushes.Black;
    }
}
```

### ConvertBack Method

```csharp
public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
{
    return value;
}
```
