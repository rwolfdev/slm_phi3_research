 # FullNameToColorConverter Class Documentation

## Purpose

The `FullNameToColorConverter` class is designed to convert a full name into a color based on the hash value of the name. It implements the `IValueConverter` interface to provide this conversion functionality.

## Class Members

### FullNameToColorConverter Class

- `Convert(object value, Type targetType, object parameter, CultureInfo culture)`: Converts the given value (full name) into a color.
- `ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)`: Returns the original value (full name) as it is.

### ColorScale Class

- `HslToRgb(double h_original, double s_original, double l_original)`: Converts HSL color values to RGB color values.

## Public Methods

### Convert Method

Parameters:
- `value`: The full name to be converted into a color.
- `targetType`: The target type, which is not used in this method.
- `parameter`: The converter parameter, which is not used in this method.
- `culture`: The culture info, which is not used in this method.

Returns:
- A `SolidColorBrush` object representing the color based on the hash value of the full name.

### ConvertBack Method

Parameters:
- `value`: The value to be converted back to a full name.
- `targetType`: The target type, which is not used in this method.
- `parameter`: The converter parameter, which is not used in this method.
- `culture`: The culture info, which is not used in this method.

Returns:
- The original value (full name) as it is.

## Public Methods Details

### Convert Method

1. Checks if the value is not null.
2. Calculates the hash value of the full name.
3. Converts the hash value to an HSL color value.
4. Converts the HSL color value to an RGB color value.
5. Creates a `SolidColorBrush` object with the RGB color value.

### ConvertBack Method

1. Returns the original value (full name) as it is.

## Public Methods Summary

- `Convert(object value, Type targetType, object parameter, CultureInfo culture)`: Converts a full name into a color based on the hash value of the name.
- `ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)`: Returns the original value (full name) as it is.

public object Convert(object value, Type targetType, object parameter, CultureInfo culture)
{
    if (value != null)
    {
        string name = value as string;

        int hash = 0;

        for (int i = 0; i < name.Length; i++)
        {
            int ascii = System.Convert.ToInt16(name[i]);
            hash = ascii + ((hash << 5) - hash);
        }

        hash = hash % 360;
        int h = hash % 360;
        int s = 45;
        int l = 60;

        return new SolidColorBrush(ColorScale.HslToRgb(h, s, l));
    }
    else
    {
        return Brushes.Black;
    }
}

public object ConvertBack(object value, Type targetType, object parameter, CultureInfo culture)
{
    return value;
}
